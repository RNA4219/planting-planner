# v0.4.1 天候 – 仕様

## 中間 API
GET /api/weather?lat=&lon= →
{
  "daily": [ { "date":"2025-10-15","tmax":22.1,"tmin":14.3,"rain":7.2,"wind":4.5 } ],
  "fetchedAt":"2025-10-15T01:23:45Z"
}
- サーバは外部 API を 24h キャッシュ

## UI
- 「天気」タブ: 日別行カード（アイコン/数値）
- ツールチップに単位・最終取得時刻

## バックオフ
- 2s → 8s → 32s（上限 60s）。失敗継続でも前回値と注記を維持
